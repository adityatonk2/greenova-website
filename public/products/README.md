# Product Images

Place your product images here with the following naming convention:

- `di-model.png` (or .jpg) - For DI Model
- `si-model.png` (or .jpg) - For SI Model
- `bmw-model.png` (or .jpg) - For BMW Model
- `fh-big-activa.png` (or .jpg) - For FH Big Activa
- `e4-model.png` (or .jpg) - For E4 Model
- `ola-model.png` (or .jpg) - For OLA Model
- `vespa-3.png` (or .jpg) - For Vespa 3

**Recommended image specifications:**
- Format: PNG or JPG
- Aspect ratio: 4:3 or 16:9
- Minimum size: 800x600px
- Maximum size: 2000x1500px
- File size: Under 500KB per image (optimized)

The images will automatically appear on the website once you place them in this folder with the correct filenames.
